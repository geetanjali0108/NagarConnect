# NagarConnect – Secure Vehicle Tracking & Monitoring System

## Overview
NagarConnect is a MERN-stack application that enables secure real-time vehicle tracking and monitoring. It provides live location updates, route visualization, and user authentication for efficient fleet management.

## Features
- Real-time vehicle tracking
- Live location updates using Socket.io
- Google Maps API integration
- JWT-based user authentication
- Route visualization
- RESTful APIs
- Responsive user interface

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB
- Socket.io
- Google Maps API
- JWT Authentication
